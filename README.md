# ProfitPulse – AI-Assisted Business Log Analyzer

ProfitPulse is an AI-powered iOS application that analyzes raw operational business logs and transforms them into structured financial insights such as revenue, cost, profit, risk score, and future projections.

The app integrates OpenRouter LLM APIs to convert unstructured log data into actionable metrics and presents them through a professional financial dashboard built with SwiftUI and Swift Charts.

---

## 🚀 Features

- AI-powered log analysis using OpenRouter API
- Structured financial metric extraction
- Revenue, Cost, Profit & Margin calculation
- Failure rate and operational risk scoring
- Monthly performance estimation
- Forecasted next-month revenue prediction
- Professional vertical bar chart dashboard
- Graceful fallback with mock data if API fails
- MVVM architecture
- Dependency Injection using DI container
- Built with Swift Concurrency (async/await)

---

## 🛠 Tech Stack

- Swift 5
- SwiftUI
- Swift Concurrency (async/await)
- URLSession
- Swift Charts
- OpenRouter LLM API
- MVVM Architecture
- Dependency Injection
- Git & GitHub

---

## 📊 What the App Does

1. User pastes business operational logs
2. Logs are sent to OpenRouter LLM
3. AI extracts structured financial metrics
4. Metrics are parsed and validated
5. Results are displayed in a professional dashboard
6. If API fails or rate limits occur, demo data is shown automatically
